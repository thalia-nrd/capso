# cabinna
A cozy digital frame where users can personalize their own space 🌸 add photos, notes, trinkets, and memories. It’s a playful and expressive web app that blends creativity with personal organization.  

---

## ✨ Features

- 🗝️ **Password Chest** – store private notes or secrets behind a password.
- 📝 **Sticky Notes & To-Do Lists** – jot down quick reminders or tasks.
- 📓 **Diary / Journal** – write entries protected by a passcode.
- 🖼️ **Photo Frames** – upload your favorite photos to display.
- 🕒 **Digital or Vintage Clock** – customizable designs.
- 🗓️ **Calendar** – keep track of days and events.
- 🎧 **MP3 Player (Spotify integration)** – for cozy background music.
- 🪞 **Magic Mirror** – shows random quotes or affirmations.
- 🪴 **Decor Elements** – drag-and-drop items like plants, books, trinkets.
- 🎨 **Custom Layouts** – users can design and save their personalized space.

---

## 🧩 Tech Stack

**Frontend**
- React + TypeScript
- Zustand (state management)
- TailwindCSS (styling)
- Framer Motion (animations)
- React DnD / drag-and-drop support

**Backend**
- Node.js + Express
- Prisma ORM + PostgreSQL
- JWT Authentication
- Cloudinary (for image uploads)

**Dev Tools**
- ESLint + Prettier + Husky
- Vite (frontend dev server)
- pnpm or npm workspaces (for monorepo setup)
